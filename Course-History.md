# 课程介绍

《大数据与机器智能》于2016年秋季开课，属于“科学课组类”通识课程。

（英文：Big Data and Machine Intelligence， 简写：BDMI）

课程中从计算原理出发，学习基本的算法与数据结构，理解数据系统（Data System）的基本原理与技术。

同时，通过学习机器学习理论，尤其是深度学习（Deep Learning）技术，理解机器智能（Machine Intelligence）的原理。
