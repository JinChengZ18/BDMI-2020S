## 目录结构

- Computing/ 计算基础与python学习
- Logistics/ 签到与团队建设
- ML-BD-Algo 算法-数据管理-机器学习材料（思路）
- Schedule 前9周教学安排
- Weeks/WWn/ 后9周教学安排，第n次课的课件和参考代码
- Course-Projects/ 课程实践任务
- Readings/ 课程课外阅读的论文材料
